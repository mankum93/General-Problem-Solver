# General Problem Solver

GPSIF is an open framework for serious AI-assisted problem solving.

It helps capable AI agents choose the right execution form, use sources responsibly, validate outputs, and preserve continuity across complex work.

See:

- docs/QUICKSTART.md
- docs/AGENT_PREREQUISITES.md
- docs/ARCHITECTURE.md
- 00_START_HERE.md

Core principle: use the lightest execution form that still preserves reasoning quality, source discipline, validation, and trustworthy conclusions.
