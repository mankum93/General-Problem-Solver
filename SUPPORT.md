# Support

For questions, use GitHub Issues once the public repository is live.

Before opening an issue, check:

- README.md
- docs/QUICKSTART.md
- docs/AGENT_PREREQUISITES.md
- docs/ARCHITECTURE.md

Good support questions include:

- Which route should be used for this task?
- Should task graph activate here?
- What validation should this workflow require?
- Is this profile wording too ceremonial or too weak?

For legal licensing questions, consult a qualified legal source. Repository docs can help compare options, but they are not legal advice.
